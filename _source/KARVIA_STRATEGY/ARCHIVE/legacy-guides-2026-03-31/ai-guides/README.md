## 💡 Key Concepts

**Claude AI Best Practices**:
- Always check MASTER_STRATEGY_DEV_TODO.md at start of session
- Use TodoWrite tool to track progress
- Reference primary source documents (MVP_STRATEGY_V5, database_schema.md)
- Update docs in place, don't create duplicates

**PDF Creation**:
- HTML files → Open in browser → Print to PDF
- Landscape layout, minimum margins, background graphics ON
- Save with clear naming: KARVIA_Pro_[Title].pdf
