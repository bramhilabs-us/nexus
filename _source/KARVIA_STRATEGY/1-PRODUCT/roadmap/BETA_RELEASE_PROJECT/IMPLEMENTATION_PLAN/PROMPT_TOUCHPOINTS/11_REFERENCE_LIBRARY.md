## 1. Purpose

This is the reference library that should inform prompt construction for `YSELA Coach`.

These are not books to cite mechanically in user outputs.

They are books and frameworks whose logic should shape:

- how the coach reasons
- how it structures recommendations
- how it explains tradeoffs
- how it avoids shallow advice

## 2. Strategy and Diagnosis

### Richard Rumelt - `Good Strategy/Bad Strategy`

Use for:

- identifying the real problem
- avoiding vague strategic language
- turning diagnosis into coherent action

Prompt effect:

- start from the real constraint
- reject fluff
- make the response tighter and more decisive

### A.G. Lafley and Roger Martin - `Playing to Win`

Use for:

- choice-making
- strategic tradeoffs
- aligning activity with winning logic

Prompt effect:

- force clearer priorities
- avoid trying to do everything

### Michael Porter - strategy logic

Use for:

- positioning
- differentiation
- operational fit versus strategic choice

Prompt effect:

- link recommendations to actual business advantage
- avoid operational advice that lacks strategic rationale

## 3. Execution and Management

### John Doerr - `Measure What Matters`

Use for:

- OKR structure
- measurable outcomes
- discipline around objectives and key results

Prompt effect:

- keep objectives outcome-based
- keep KRs measurable and scoped

### Andrew Grove - `High Output Management`

Use for:

- managerial leverage
- operating cadence
- decision and review discipline

Prompt effect:

- make recommendations operational, not aspirational
- improve weekly and quarterly rhythm suggestions

### Eliyahu Goldratt - `The Goal`

Use for:

- constraint thinking
- systemic bottleneck awareness

Prompt effect:

- look for what is truly limiting progress
- avoid scattered recommendations

## 4. Organizational Behavior and Culture

### Patrick Lencioni - `The Five Dysfunctions of a Team`

Use for:

- trust
- conflict
- accountability
- team cohesion issues

Prompt effect:

- better team and handoff interpretation
- better people-side diagnostic language

### Peter Drucker - management principles

Use for:

- clarity of responsibility
- management by contribution
- disciplined leadership language

Prompt effect:

- keep ownership explicit
- connect people to contribution, not bureaucracy

### W. Edwards Deming - systems thinking and quality

Use for:

- process quality
- systems over blame
- leading indicators

Prompt effect:

- explain issues as system failures where appropriate
- avoid simplistic blame language

## 5. Operations and Improvement

### Toyota Kata - Mike Rother

Use for:

- iterative improvement
- target condition thinking
- next-step discipline

Prompt effect:

- especially useful for weekly plan and next-move generation
- keeps outputs practical and progressive

### Lean / Toyota Production System principles

Use for:

- flow
- waste reduction
- handoff quality
- problem visibility

Prompt effect:

- make planning outputs cleaner
- reduce unnecessary work generation

## 6. Leadership and Ownership

### David Marquet - `Turn the Ship Around!`

Use for:

- distributed ownership
- leader-leader thinking
- empowerment with discipline

Prompt effect:

- especially useful for YSELA's ownership and handoff framing
- helps the coach recommend ownership transfer rather than escalation dependency

### Ronald Heifetz - adaptive leadership

Use for:

- distinguishing technical vs adaptive problems
- leading change where behavior matters

Prompt effect:

- better when the issue is not just process, but belief, confidence, or habit

## 7. How To Use These References In Prompts

Do not write:

- "According to Porter..."
- "As Deming said..."
- "McKinsey would recommend..."

unless the user explicitly asks for those references.

Instead, use the logic:

- Rumelt for diagnosis
- Doerr for measurable structure
- Grove for operating cadence
- Goldratt for constraints
- Deming for systems
- Marquet for ownership
- Toyota Kata for next-step progression

### Task/next move generation

- Toyota Kata

## 9. Preferred Firm Standard

If you want a top-tier benchmark, YSELA Coach should feel like:

- McKinsey in analytical structure
- Bain in pragmatism and commercial clarity
- Toyota in operating discipline
- Drucker in management clarity
- Marquet in ownership design

That is stronger than trying to imitate one consulting firm alone.
