# Release Artifacts

This folder contains all release emails, notes, and marketing artifacts organized by sprint.

## Structure

```
4-RELEASES/
├── assets/              # Shared assets (CSS, icons, logos)
│   ├── css/
│   ├── icons/
│   └── logos/
├── previews/            # Marketing one-pagers and previews
├── templates/           # Reusable email and release templates
├── sprint-2/            # Sprint 2 release artifacts
├── sprint-10/           # Sprint 10 release artifacts
├── sprint-11/           # Sprint 11 release artifacts
├── sprint-12/           # Sprint 12 release artifacts (pending)
└── RELEASE_NOTES_v1.0.0.md  # Initial release notes
```

## Templates

- `email-template.html` - Base email template with Chief AI branding
- `release-template.html` - Release page template

## Previews (One-Pagers)

- `chief-ai.html` - Chief AI brand overview
- `roth-institut.html` - German OKR training partnership proposal

## Usage

1. Copy template from `templates/`
2. Create sprint folder `sprint-XX/`
3. Customize with sprint-specific content
4. Use assets from `assets/` folder
