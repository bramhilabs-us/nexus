# Consultant Onboarding Narrative: From SSI Insight to Daily Execution

This blog explains the consultant journey at application level, using one clear flow:

Assessments -> Choose focus -> Build teams -> Align on objectives -> Plan journey -> Execute daily tasks

## Why this journey matters

Consultants need a repeatable way to move clients from diagnosis to action.  
Karvia provides that operating rhythm by connecting SSI-based insight with objective setting and daily execution.

## Step 1: Assessments and focus choice

**Page objective:** Create a shared baseline for Speed, Strength, and Intelligence.  
**What consultants do:** Launch assessments, explain SSI interpretation, and guide stakeholders to choose one priority gap.

**Page objective:** Keep every team focused on today's work.  
**What consultants do:** Coach managers to use the task board daily for updates, reassignment, and completion flow.

![Team Task Management](/Users/sagarrs/Desktop/official_dev/karvia_business/docs/consultant-onboarding/assets/screenshots/team-tasks.png)

## Consultant facilitation model

Use the same three prompts in every client workshop:

1. What does SSI say is the real constraint?
2. Which objective will reduce that constraint fastest?
3. What daily behaviors prove we are moving?

This keeps onboarding practical, outcome-focused, and measurable.

## Final message to onboarded consultants

Do not start from goal templates.  
Start from SSI evidence, choose priorities with the client, build the right team, then execute through daily task rhythm.
